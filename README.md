# phpencrypt - PHP代码加密

用于加密php文件，虽然自己也喜欢开源，but为了生活还是要做些妥协的...

## 使用说明

	-d	file or directory
	example:
	php encrypt.php -d D:\test\a.php
	php encrypt.php -d D:\test\
	php encrypt.php -d /home/www/index.php

会自动将加密文件或文件夹备份到bak目录，所以不用担心，自己可以对test文件夹测试一下看看
